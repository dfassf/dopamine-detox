## 1. UX 개선

### 1-1. 입력 UX 개선
- 폼 입력 시 엔터/완료 누르면 다음 항목으로 자동 포커스 이동
- 숫자 입력 필드에 `inputMode="numeric"` 적용
- 모바일 키보드 "다음" 버튼으로 항목 간 이동 가능하도록
- 마지막 항목에서 "완료" 시 제출 버튼 포커스

### 1-2. 회원가입 후 온보딩 플로우 변경
현재: 회원가입 → 대시보드 (빈 화면)
변경: 회원가입 → **디톡스 설정 화면** (온보딩)

```
[회원가입 완료]
    │
    ▼
[온보딩: 디톡스 시작하기]
    │
    ├─ "시작하기" → 디톡스 생성 폼 (/abstinence/new)
    │
    └─ "둘러보기" → 대시보드 (/dashboard)
```

- 신규 사용자에게만 표시 (디톡스가 0개일 때)
- "둘러보기" 선택 시 대시보드로 이동 (빈 상태 + "디톡스 시작하기" 버튼)

## 2. AI 타임라인 활성화

### 2-1. Cloud Run 환경변수 설정
```bash
gcloud run services update dopamine-detox \
  --project personal-projects-ss \
  --region asia-northeast3 \
  --set-env-vars "GEMINI_API_KEY=<키값>"
```

### 2-2. AI 타임라인 생성 흐름 (이미 구현됨, 키만 넣으면 동작)
- 금욕 등록 시 → Gemini API 1회 호출 → 개인화된 타임라인 생성
- 주간 체크인 시 → Gemini API 1회 호출 → 남은 타임라인 재조정
- API 키 없으면 → 템플릿 기반 fallback (현재 상태)

### 2-3. 타임라인 데이터 정교화
- 금연 타임라인 (smoking.json) 의학 근거 보강
- 식단 타임라인 (diet.json) 구체화
- 커스텀 타임라인 행동과학 메시지 추가
